
# Sports frontend web

An application that retrieves the sports sessions that the user has performed




## Installation
Install Sports_frontend_web with npm

```bash
git clone https://github.com/BlandineM/Sports_frontend_web
cd ../path/to/the/file
npm install
npm start
```
